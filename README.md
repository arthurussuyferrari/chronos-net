⏱️ Chronos Net

Chronos Net é uma extensão para Google Chrome que monitora e gerencia o tempo gasto em sites de forma automática e transparente.
Ela registra quanto tempo o usuário permanece ativo em cada domínio, permitindo uma visão clara dos hábitos de navegação e ajudando na melhoria de foco, produtividade e uso consciente da internet.

A extensão funciona em segundo plano, detectando quando uma aba está ativa ou inativa, acumulando o tempo real de uso por site e armazenando esses dados localmente no navegador, sem dependência de serviços externos.

Principais objetivos

Monitorar o tempo real gasto em cada site

Agrupar dados por domínio

Armazenar informações localmente com segurança

Fornecer estatísticas simples e acessíveis via popup

Ajudar o usuário a entender e gerenciar melhor seu tempo online

Tecnologias

TypeScript

Chrome Extensions (Manifest V3)

Service Workers

Content Scripts

Chrome Storage API

Privacidade

O Chronos Net não coleta, envia ou compartilha dados externos.
Todas as informações permanecem exclusivamente no navegador do usuário.
